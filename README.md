# networks
Задания по курсу компьютерных сетей

## `RIP` Алгоритм
Запуск
```
go run main.go <n> <prob>
```
`n` - количество маршрутизаторов

`p` - вероятность, что между двумя маршрутизаторами будет ребро

- (Если выбрать `p` маленьким может получится несвязный граф)

Eсли у маршрутизатора не будет пути к другому, то он не будет ничего писать в своей таблице про него.

Каждый маршрутизатор обрабатывает запросы асинхронно.
